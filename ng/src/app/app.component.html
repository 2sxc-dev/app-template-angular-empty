App Component – Empty Angular App

This is an empty Angular app.

Getting Started

  1. Install dependencies: 
    npm install
    Run this command in the root directory of your project to install all required packages.
  2. Adjust output folder:

    If you want to publish the app, adjust the outputPath in the angular.json file.
    Example: 

    "outputPath": "dist/your-directory"
  3. Start the app locally (with SSL): 
    ng serve --ssl true
    Then open https://localhost:4200 in your browser.
    If you see a security warning, click “Advanced” and accept the certificate (“Proceed to unsafe site”).
